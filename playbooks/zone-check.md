# ゾーン確認プレイブック

実践中にゾーンを確認するときの手順。

## 確認順

1. **現在G** を確定する（曖昧ならユーザーに確認）
2. 機種ファイルのゾーン表を見る（`knowledge/machines/`）
3. 直近のボーナス／CZ終了地点から「どの帯にいるか」を特定
4. 天国・通常・地獄などのモード仮説を立てる
5. 「今打つ理由」または「様子見理由」を1行で言う

## 返答で必ず分けるもの

| ラベル | 意味 |
|--------|------|
| 確定 | スペック上ほぼ疑いようがない |
| 有力 | 状況から濃厚だが確定ではない |
| 可能性 | 候補のひとつ |
| 不明 | 材料不足 |

## よくある失敗

- G数未確認のままゾーン断定
- モードとゾーンを混同
- 設定示唆とゾーンを同一視

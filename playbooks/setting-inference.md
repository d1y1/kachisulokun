# 設定・期待値の読みプレイブック

実践共有から設定の期待値や継続判断を整理する手順。

## 確認順

1. 目的を確認（設定狙い / ゾーン消化 / イベント拾い / 射幸心だけ）
2. 機種の「設定差が付きやすい要素」を机種ファイルから取得
3. ユーザー報告を材料リスト化（示唆・挙動・出玉・モード）
4. 材料を **確定 / 有力 / 可能性 / 不明** に分類
5. 結論は「期待値の向き」まで。設定番号の断定は材料が揃ったときだけ

## 期待値の言い方

- 「プラス期待寄り」「ほぼトントン」「マイナス寄り」
- 可能なら「根拠が何個あるか」を添える
- 店舗傾向がある場合は設定判別と分けて書く

## ヤメ判断の型

```
続ける: 根拠が残っている / 次の確認地点が近い
様子見: 材料不足、あとNゲームで分岐
切り上げ: 否定材料が強い / 機会費用が大きい
```

## 注意

- 単発のイイ感じ演出だけで高設定断定しない
- 「負けている」こと自体は設定の証拠にしない
- 不明なら不明と書き、追加で見るポイントを出す

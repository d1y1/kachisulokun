# PIA横浜モアーズ店

- エリア: 神奈川県横浜市西区南幸1-3-1（JR横浜駅西口・モアーズ地下）
- 最終更新: 2026-07-13
- スラッグ: `pia-yokohama-mores`

## 基本

- 台数規模: パチンコ537台 / スロット543台（神奈川屈指の大型）
- 主な機種帯: 北斗・エヴァ・からくり等の人気機種を大量設置。駅前地下フロア
- 営業時間: 9:00〜22:45
- 換金: 等価（21.73円・10円スロ / 4.31円・1円パチ）
- 再プレイ: 無制限
- 特徴: 123横浜西口と合計1080台。県内で「強い店」の筆頭候補

### 入場

## イベント傾向

### 確定（みんパチ・公開情報）

- **旧イベント日**: 1のつく日 / 5のつく日 / 毎月11日・22日 / 月と日がゾロ目の日
- **周年**: 3月15日（2005年オープン）
- ※7のつく日は**公式旧イベではない**（ブログ等の誤記に注意）

### 日付別公約（X・口コミ — 要検証）

| 日付 | 公約内容（可能性） |
|------|-------------------|
| 1のつく日 | 人気機種への散らし・並び |
| 5のつく日 | 口コミで「123が控えめでPIAが強い」報告 |
| 11・22日 | 楽園系と同系統の旧イベ。全台系は日による |
| ゾロ目 | 月日ゾロ目は年イチ級の期待値 |

### 強い日の仮説（データベース）

- **5のつく日**: 口コミで利益狙いの日として言及多い
- **11/22日・ゾロ目**: 旧イベとしてデータ蓄積あり
- **3/15 周年**: 年イチ級投入の可能性

### 弱い日の仮説

- 平常日でも稼働は高いが、広いフロアで当たりが散らばる
- 旧イベでも日によってマイナス履歴あり → 全台高設定ではない

## 立ち回りメモ

### 朝イチ

- 8:30抽選。旧イベ・周年は7:00台到着も検討
- 入場後は**稼働上位の人気機種島**から
- 抽選負け時は**アビバ関内**等をサブ候補に

### 後ヅモ

1. みんレポで総差枚・機種別
2. G2000前後まで積み上がった台を優先
3. 2台設置は並び単位で判断

### 撤退

- 広い店なので移動コスト大。浅い台は早めに切る
- 旧イベでも全台勝ちではない

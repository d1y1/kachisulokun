# 店舗ナレッジ

通う店舗の傾向メモ。

- 新規は `_template.md` をコピーして `<店舗スラッグ>.md`
- 個人的仮説は「仮説」「可能性」と明記する
- **情報源（X・みんレポ等）・台番末尾・イベント結果**を必ずセクション分けする

## 店舗提案の優先度（エージェント向け）

**データが厚い店を、よく行く店より上に置かない。**

| 優先度 | 区分 | 店舗 | 扱い |
|--------|------|------|------|
| **1** | よく行く店 | ベルシティ元住吉・マルハン鶴見・PIA綱島 | 店舗提案の**第一候補**。旧イベが合わなくてもここから組み立てる |
| **2** | たまに行く店 | アビバ綱島南・アビバ綱島樽町 | カレンダー・データが合う日だけ候補に入れる。無理に最優先にしない |
| **3** | 遠征・候補 | その他の登録店 | 旧イベ・取材・ユーザーが遠征OKと言ったとき |

- 各店舗ファイルの `訪問頻度` フィールドを参照
- `playbooks/seat-number-analysis.md` のアビバ綱島南は**分析例**。提案時のデフォルト店舗ではない

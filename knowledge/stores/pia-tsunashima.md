# PIA綱島

- エリア: 神奈川県横浜市港北区綱島西2-2-22（東横線綱島駅西口徒歩1分）
- 最終更新: 2026-07-13
- スラッグ: `pia-tsunashima`

## 基本

- 台数規模: パチンコ244台 / スロット203台
- 主な機種帯: **綱島駅前でパチンコが打てる唯一の店**（P-WORLD）。ジャグラー列＋バラエティ
- 営業時間: 9:00〜22:45
- 換金: 等価（21.73円スロ）
- 再プレイ: 無制限

### 入場

| 時刻 | 内容 |
|------|------|
| 8:20 | 先着順で整理券配布（1階マンション側通路） |
| 8:55 | 再整列 |
| 9:00 | 入場 |

- 会員カード不要
- 駅前1分のため**朝は並びが早い** → 7:30台到着も検討

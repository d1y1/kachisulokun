# アビバ関内店

- エリア: 神奈川県横浜市中区真砂町3-33（関内駅・桜木町駅方面）
- 最終更新: 2026-07-13
- スラッグ: `aviva-kannai`

## 基本

- 台数規模: パチンコ約550台 / スロット約560台（合計1111台級の大型）
- 主な機種帯: 最新台・バラエティ豊富。AT＋ジャグラー＋1台バラエティ
- 営業時間: 9:00〜22:45
- 換金: 等価
- 再プレイ: 無制限
- 特徴: 横浜駅激戦区を避けた**玄人向け穴場**候補。機種数が多い

### 入場

| 時刻 | 内容 |
|------|------|
| 8:30 | 抽選 |
| 9:00 | 入場 |

- 会員カード不要
- PIAモアーズ旧イベ日は**並びが横浜駅より落ち着く**可能性

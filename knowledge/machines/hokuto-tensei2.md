## シャッター（ゾーン外前兆）

特定帯でシャッター／ガセ前兆あり → **896あべし以内当選が有力**（設定示唆ではなくモード寄り）。

| 帯（あべし目安） | 扱い |
|------------------|------|
| 32〜48（実戦40前後） | 第1帯。AT後の最重要判別 |
| 160〜176 | 第2帯 |
| 288〜304 | 第3帯 |
| 416〜432 | 第4帯 |
| 480〜496 | 第5帯 |

- あり → 次回ATまで継続
- なし（特にAT後45前後まで）→ 896貫通（モードA）疑い → ヤメ検討
- レア役契機のシャッターは除外。雷鳴のみ等は前兆扱い注意

## 実践チェックリスト

- [ ] あべし数（G数だけで決めない）
- [ ] ランプ位置（A／B／C）と色・点灯or点滅
- [ ] 天撃失敗（表示あべしが赤など）→ フォロー必須
- [ ] AT直後か／天国否定（128超え）か
- [ ] 伝承モード・天破経由の有無
- [ ] 有利区間差枚（1500〜1600枚帯のあとづも説あり）

## ヤメどころ目安

- AT終了後、天破経由なし・前兆なしなら即ヤメ寄り
- 天国否定（128超え）で材料なしならヤメ
- 天撃失敗後はヤメない
- 256帯消化後に材料なしなら区切り
- **42未確認＋176帯シャッターなし** → サイド青点灯（896弱）だけなら256確認後ヤメでよい（2026-07-19鶴見実戦）

## 実践メモ（ユーザー実戦）

- 即やめ台でも **42帯シャッター確認できれば続行**（896以内有力）
- 典型フォロー: **42前後 → 160〜176 → 193〜256**。2段ヒット後の256帯当選を複数回確認
- 天破経由で42帯を通せない周期は、176帯の有無を次の分岐にする
- シャッター的中でも出玉が薄い周期は設定材料にしない（2連299／3連418級は粘り材料薄）

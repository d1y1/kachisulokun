# からくりサーカス

- メーカー: SANKYO
- 導入時期: 2023年7月
- 最終更新: 2026-07-14

## スペック要約

- タイプ: スマスロAT
- 純増: 約2.8枚/G（通常時）/ 約7.6枚/G（AT中）
- 天井: CZ間1200G / CZ4スルーでAT / AT間2500G
- リセット挙動: 天井G数・CZスルー回数リセット
- 設定示唆の有無: あり（終了画面・連打上乗せ・ランプ色）

## ゾーン（モード別天井）

| モード | 最大天井 | 備考 |
|--------|----------|------|
| 通常A | 1200G（実平均730G） | 100の位偶数がチャンス |
| 通常B | 800G（実平均550G） | 次回通常C以上期待度UP |
| 通常C | 1200G（実平均730G） | G数当選でAT直撃 |
| 天国 | 100G（実平均70G） | 100G以内CZ以上当選 |

## 設定差が付きやすいポイント

- 終了画面のキャラ
- 連打上乗せ表記
- エンディングランプ色

## 実践チェックリスト

- [ ] CZ間G数（液晶表示）
- [ ] CZスルー回数（3スルー以上は即打ち）
- [ ] 前回当選G数（1100G以上なら短縮狙い）
- [ ] スポットライト（規定G数示唆）

## ヤメどころ目安

- AT終了後: 幕間チャンス後1G回し → ワイプ確認
- 天国フォロー: 100Gまで
- CZスルー3回なら次回CZまでフォロー推奨

## メモ・出典

- CZスルー天井は「機械仕掛けの神」のみカウント。幕間チャンスはノーカン
- 上位AT後の天国移行率約40%は狙い目

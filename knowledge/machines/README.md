# 機種ナレッジ

実践で使う機種の基本情報を置く。

- 新規は `_template.md` をコピーして `<機種スラッグ>.md` で保存
- ファイル名例: `myjuggler5.md`, `go-go-juggler3.md`
- 不明な項目は空欄か「不明」のまま残す

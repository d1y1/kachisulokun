# スマスロ ビッグドリーム THE GOLDEN PUSHER

- メーカー: サミー（銀座）
- 導入時期: 2026年5月11日
- 最終更新: 2026-07-19

## スペック要約

- タイプ: スマスロAT（メダルプッシャー融合）
- 純増: 約8.2枚/G（ゴールデンボーナス）
- コイン持ち: 約30.7G/50枚
- 天井:
  - 通常（AT後）: 最大1499GでAT（333 / 555 / 999 / 1499 から選択）
  - 設定変更時・上位AT後: 最大999Gに短縮（333 / 555 / 999）
  - 別軸: **ボール天井**（規定個数落下でプレミアムCZ系恩恵）
- リセット挙動: 設定変更でゲーム数天井が999G以下に短縮。内部状態もリセット
- 設定示唆の有無: あり（AT終了画面・アイキャッチ・サミートロフィー・EDボイス等）

## ゲーム性の要点

- 通常時はプッシャー＋メダル落下。**ボール落下 → CZ（3段階ジャッジ）**
- 最終CZ「ジャッジメント」突破でAT「ゴールデンボーナス」
- AT突入時は初期枚数ゾーン「ドリームラッシュ」（保証上乗せ3回）
- プレミアムCZ成功 → プレミアムAT経由で上位AT「超ゴールデンボーナス」
- AT/上位AT終了後は**引き戻し約77G**（通常AT後はCZ高確寄り、上位AT後はAT高確寄り）

## ゾーン・天井帯

| 区間 | 内容 | 備考 |
|------|------|------|
| 50〜250G | 当選薄め（実戦値） | Gゾーン単体の厚みは弱い |
| 333G | 仮天井候補 | 選択率自体は低め。浅天井示唆なら強い |
| 555G | 仮天井候補 | 実戦値ではやや期待できる帯 |
| 999G | 天井候補 / リセ・上位AT後の最大 | |

- 本機は「固定Gゾーン狙い」より **天井選択・ボール天井・赤/金ボール** が立ち回りの核

## 狙い目（等価・目安）

| 条件 | 目安 | 確度 |
|------|------|------|
| 通常AT後天井 | 約400G〜（現実的には500G前後〜） | 有力 |
| リセ / 上位AT後 | 約200G〜（現実的には300G前後〜） | 有力 |
| 赤ボール近い | 落下までフォロー | 有力（1st突破約80%） |
| 金ボール表示 | 落下まで必須 | 有力（プレミアムCZ濃厚寄り） |

※5.6枚交換は通常AT後で750G前後〜が無難な目安（解析サイト概算）

## ヤメどころ目安

1. AT終了後 → **引き戻し約77G**を確認。非当選ならヤメ目安
2. 液晶に**赤/金ボール**があれば落下まで続行
3. 「JUDGEMENT」保留が出たら消化までフォロー
4. ボール天井示唆ボイス（テラ〜／カウントダウン系）が濃いなら継続検討

## アイキャッチ（CZ/AT終了時）

| 種類 | 示唆 | 強さ |
|------|------|------|
| 通常 | デフォルト | 弱い |
| 赤 | 天井999G以下のチャンス（CZ終了時ならさらにアツい） | 強 |
| キリン柄 | 天井333G濃厚＋ボール天井到達済み | 最強 |

補足:
- 上位AT後の**初回**は赤アイキャッチが基本。ここで**通常**が出ると法則崩れ→天井555G以下の大チャンス
- 初回以降の通常は法則崩れにならない

## 設定差が付きやすいポイント

- **AT初当り**（CZより差が大きい）
- **天井ゲーム数選択**（1499G以外＝高設定寄り）
- AT終了画面・サミートロフィー（上表）
- アイキャッチ（上表）
- ステーションチェンジ時の初期ボール数（高設定ほど多め寄り・差は小さめ）

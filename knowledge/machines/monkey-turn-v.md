# スマスロモンキーターンV（モンキーターン5）

- メーカー: 山佐（YAMASA）
- 導入時期: 2023年12月4日
- 最終更新: 2026-07-14
- 型式名: Lモンキーターン5CE

## スペック要約

- タイプ: スマスロAT
- 純増: 約2.5枚 or 4.0枚/G（上位ATあり）
- コイン持ち: 約32G/50枚
- 天井:
  - ゲーム数: 795G+α（リセット・青島VS波多野敗北後は 495G+α）
  - 周期: 最大6周期（同上短縮時は最大4周期）
- リセット挙動: 天井短縮、1周期目は222pt以内で優出モード
- 設定示唆の有無: あり（ヘルメット・舟券・終了画面等）

## あとづも（ハイエナ）要点

離席直後にまず見るもの:

1. AT終了画面のヘルメット
2. サブ液晶のライバルメット / EXアイテム
3. 現在G・周期・激走pt
4. 有利区間差枚（目安 +1400枚以上）

### 即乗り（AT当選まで）

| 条件 | 確度 | 備考 |
|------|------|------|
| ヘルメット ロゴ+キラキラ | 有力 | 通常B以上濃厚 |
| ヘルメット ロゴ+V | 有力 | 天国濃厚 |
| サブ液晶 青島 / 浜岡 / モノクロ波多野 | 有力 | ライバルモード恩恵大 |
| EXペンギンストック所持 | 確定寄り | 青島SGスタート |
| AT後1Gで弱チェ・ボート・弱チャンス目 | 有力 | 通常B以上 → 次回ATまで |
| 青島VS波多野敗北直後 | 有力 | 天井短縮（495G / 4周期） |

### 1周期フォロー寄り

| 条件 | 確度 | 備考 |
|------|------|------|
| ヘルメット ロゴあり | 可能性 | 通常B期待度アップ |
| サブ液晶 榎木 | 可能性 | 優出期待度50%以上 → 周期まで |
| 有利区間差枚 +1400〜1600枚でAT終了 | 可能性 | 有利切れ→青島VS波多野狙い |
| ステチェン黒背景 | 可能性 | 当該周期チャンス → 周期まで |

### 見送り寄り

- ヘルメットロゴなし・ライバルデフォ（波多野）・周期浅め・G浅め・アイテムなし
- 基本は AT後1G回して上記が無ければヤメ

## 天井狙い目目安（解析サイト平均）

| 条件 | 等価目安 | 備考 |
|------|----------|------|
| 通常（据え寄り） | 410G〜 または 5周期〜 | 周期・pt・示唆で調整 |
| リセット / 青島VS後 | 50G〜 | 0Gでも出率は乗せやすいが、少し回ってからが無難 |
| 5.6枚現金 | 500G〜 | 店舗交換率に合わせて深く |

出典によりボーダー差あり（350〜450G帯の記事も）。**周期・ヘルメット・アイテムを見て上下する**前提。

## 設定差が付きやすいポイント

- 初当たり確率
- AT継続・上位AT関連
- ライバルモード・終了画面系の示唆出現率

## 実践チェックリスト

- [ ] 現在G（データカウンタ）と周期数・激走pt
- [ ] AT終了ヘルメット（ロゴなし / ロゴ / キラキラ / V）
- [ ] サブ液晶: ライバルメット・舟券・EXアイテム
- [ ] 青島VS波多野履歴（ゲーム数ズレ・履歴に10/11G並び）
- [ ] 有利区間差枚目安（+1400枚前後）

## ヤメどころ目安

- 基本: AT終了後 → ヘルメット確認 → 1G回してヤメ（即優出カバー）
- 続行: 天国・通常B濃厚示唆、青島/浜岡/モノクロ波多野、ペンギンストック、有利切れ寸前

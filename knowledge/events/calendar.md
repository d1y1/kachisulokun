# 旧イベ日付カレンダー（日にち × 候補店）

「今日どこ打つ？」の即答用ベースライン。
**これはあくまで旧イベの静的マップ。当日予想では最新X（店舗公式＋情報垢）を上書きで効かせる**（`playbooks/information-sources.md`）。

- 出典: `knowledge/stores/README.md` の各店旧イベ（確定/有力）。店名変更・優先度変更時はこの表も更新する
- 「のつく日」はこのリポでは**末尾ベース**で扱う（例: 1のつく日 = 1・11・21・31日。15日は含めない）
- 30日・31日の扱いが店により分かれる場合は各店舗ファイルを優先

## 使い方

1. 当日の日にちで行を引く → 優先度1から候補を出す
2. 優先度1が空欄（2・10・20日など）の日は「平常日の散らし狙い」か優先度2のカレンダー一致を検討
3. **必ず最新Xで上書き**（取材・全台系告知があれば旧イベ外でも昇格。7/15のM&M昇格が前例）
4. 答え合わせは `playbooks/x-event-analysis.md` セクション5

# 店舗・イベント調査

## 手順

1. `knowledge/stores/` と `knowledge/events/` を確認
2. `playbooks/store-scouting.md` に沿って整理
3. 当日情報が必要なら Web 検索（店舗公式・公開情報のみ）。推測は分けて書く
4. 実践向けに「今日の仮説」と「観測ポイント」を短く返す

## 出力テンプレ

```markdown
## 店舗仮説
- ...

## イベント読み
- ...

## 狙い目
1. ...

## 観測ポイント
- ...

## 注意
- 未確認情報はここに明示
```

## 記録

新しい傾向は `knowledge/stores/<店舗>.md` または `knowledge/events/` に追記する。

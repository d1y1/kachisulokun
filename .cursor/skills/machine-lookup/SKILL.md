# 機種情報ルックアップ

## 手順

1. `knowledge/machines/` を検索し、該当ファイルを読む
2. 無ければ `_template.md` から新規作成し、分かる範囲だけ埋める
3. 公式スペック補完が必要なら Web 検索し、出典を添える
4. 応答は「実践で使う要点」優先（長いスペック表の羅列は避ける）

## 要約フォーマット

```markdown
## 機種: <名前>

### まず押さえる点
- 天井:
- ゾーン:
- 設定差が付きやすいポイント:

### 実践チェック
- 見る示唆:
- ヤメどころ目安:

### 詳細
（必要なら knowledge ファイルへの参照）
```

## 更新

新しい知見が出たら該当 machine ファイルに追記する。出典・日付を残す。
